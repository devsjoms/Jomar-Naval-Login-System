## Description
The **Jomar-Naval-Login-System** is a basic login application built in Python. It allows users to securely log in and manage accounts using a simple interface. This project is ideal for beginners looking to understand GUI programming in Python and basic user authentication logic.

---

## Features
- User login authentication
- User-friendly GUI interface
- Easy-to-understand Python code
- Lightweight and standalone application
